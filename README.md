# Job Search OS

An AI-native decision support system designed to augment human judgment across strategic job search.

## What It Is

Job Search OS is a modular AI system built to improve the quality, rigor, and consistency of professional job search decisions.

Unlike automation-first job tools that focus on mass applications, Job Search OS is designed around a different premise:

*AI should strengthen human decision-making, not replace it.*

This system does not autonomously apply to jobs on a candidate’s behalf.  
Instead, it functions as an intelligent reasoning layer that helps a candidate:

- discover relevant opportunities
- evaluate roles based on desirability and real fit
- generate tailored application materials grounded in truth
- support targeted outreach with strategic precision

---

## Core Capabilities

### Opportunity Discovery
Identifies and prioritizes relevant public job opportunities based on role relevance, location logic, and strategic alignment.

### Opportunity Evaluation
Scores opportunities using explicit frameworks for:
- desirability
- candidate fit
- weighted opportunity ranking

### Application Generation
Creates tailored resumes and cover letters matched to actual candidate qualifications and role requirements.

### Outreach Support
Identifies high-value contacts and generates concise, role-specific outreach messaging.

---

## System Design Principles

Job Search OS is built on five core principles:

1. Human agency remains central  
2. AI supports judgment, not blind automation  
3. Outputs must remain grounded in verified evidence  
4. Decision logic should be explicit, modular, and auditable  
5. Truthful positioning is more valuable than inflated optimization
